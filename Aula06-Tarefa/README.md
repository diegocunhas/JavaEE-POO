## Objetivos

Criar um projeto com uma página JSP que a partir do ano e o mês informados como parâmetro mostre uma tabela com o calendário do mês.


## Ferramentas

* JavaEE;
* Apache Tomcat;
* HTML/CSS;
* JSP;
